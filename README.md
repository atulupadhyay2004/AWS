# AWS Monitoring & Cost Control Project

## Overview

This project demonstrates AWS monitoring, alerting, and cost management using EC2, CloudWatch, SNS, and AWS Budgets.

## Services Used

* Amazon EC2
* Amazon CloudWatch
* Amazon SNS
* AWS Budgets
* Cost Explorer
* Linux (Ubuntu)

## Features

* Launched and configured an EC2 instance
* Monitored CPU and network metrics using CloudWatch
* Created a CloudWatch Dashboard
* Configured CPU utilization alarms
* Integrated SNS email notifications
* Set up AWS Budget alerts for cost monitoring
* Practiced AWS resource cleanup and cost optimization

## Architecture
